# 🌿 生活管理系统

一个简洁的移动端友好的个人生活管理 H5 网站。

## 功能

- 🏥 **健康管理** - 健康生活原则和习惯
- 💪 **健身管理** - 各部位训练动作和计划
- 🧘 **心灵管理** - 冥想练习和情绪管理
- 📅 **计划管理** - 目标设定和习惯追踪

## 技术栈

- 纯静态 HTML/CSS/JavaScript
- Markdown 内容管理
- GitHub Pages 托管

## 如何编辑内容

1. 编辑 `content/` 目录下的 `.md` 文件
2. 提交到 GitHub：
   ```bash
   git add .
   git commit -m "更新内容"
   git push
   ```
3. 几秒后 GitHub Pages 自动更新

## 本地预览

直接在浏览器中打开 `index.html` 即可。
