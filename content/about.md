---

## 如何使用

### 编辑内容

1. 用任意文本编辑器打开 `content/` 目录下的 `.md` 文件
2. 使用 Markdown 语法编辑内容
3. 保存并提交到 GitHub

### 提交更新

```bash
cd lf_manage
git add .
git commit -m "更新内容"
git push
```

### 查看更新

GitHub Pages 会自动部署，几秒后刷新网页即可看到更新。
